Line-break

Rendu

Ceci est un paragraphe avec un line-break normal. Il devrait couper les mots seulement aux endroits normaux sans forcer de cassures supplémentaires.

Ceci est un paragraphe avec un line-break strict. Il peut casser des mots à n'importe quel endroit.

Ceci est un paragraphe avec un line-break loose. Il minimise les occasions de casser les mots, ce qui peut rendre le texte un peu plus propre visuellement, même dans des colonnes étroites.

Code

.exemple__line-break {
                             display: flex;
                             flex-direction: column;
                             gap: 1rem;
                        }
                        
                        .break {
                             border: solid var(--color-black) 2px;
                             padding: 0.5rem;
                             background-color: #eee;
                        }
                        
                        .normal-break p {
                             line-break: auto;
                             margin: 0;
                        }
                        
                        .strict-break p {
                             line-break: anywhere;
                             margin: 0;
                        }
                        
                        .loose-break p {
                             line-break: loose;
                             margin: 0;
                        }
